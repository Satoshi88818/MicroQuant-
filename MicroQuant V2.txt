MicroQuant V2 — Microtubule Lattice Quantum Processor Simulator

Version 2.0 — Full rewrite incorporating bug fixes and enhancements over V1.

Changes from V1:

Fixed XY Hamiltonian coupling (removed erroneous .dag() on second term)

Fixed per-qubit state visualization (now uses ptrace correctly)

Replaced fragile coherence metric with l1-norm of coherence

Fixed threshold_collapse to trigger on energy variance rather than raw expectation

Added thermal bath support (temperature-dependent Lindblad rates)

Added disorder (random local fields, tunable strength)

Added collective/superradiance effects (shared bath with collective jump operators)

Added Jordan-Wigner analytic comparison for correctness validation

Added gate fidelity metrics via control pulses

""" MICROTUBULE LATTICE QUANTUM PROCESSOR SIMULATOR — V2 ===================================================== A self-contained quantum computing engine modelling a 1D lattice of coupled qubits with dipole interactions. Implements coherent evolution, entanglement generation, and threshold-based collapse as a non-standard readout mechanism. Enhancements over V1: - Corrected XY Hamiltonian (no erroneous .dag() on second coupling term) - l1-norm coherence metric (replaces unreliable single off-diagonal element) - Energy-variance threshold for collapse (physically meaningful trigger) - Thermal bath: temperature-dependent Lindblad rates (n-bar formulation) - Disorder: random local fields to mimic biological variability - Collective effects: shared bath with collective jump operators (superradiance) - Jordan-Wigner free-fermion analytics for correctness benchmarking - Gate fidelity metric via simple Rabi control pulse Requirements: pip install numpy matplotlib qutip scipy """ import numpy as np import matplotlib.pyplot as plt import qutip as qt from scipy.linalg import eigh from typing import Tuple, List, Dict, Optional # ========================== CONSTANTS ========================== N_QUBITS = 6 # Lattice size (keep ≤8 for speed; Hilbert space = 2^N) COUPLING_J = 2.0 # XY nearest-neighbour coupling strength LOCAL_COUPLING = 1.0 # Local dipole coupling scale DECOHERENCE_RATE = 0.05 # Base amplitude-damping rate γ COLLAPSE_THRESHOLD = 2.0 # Energy-variance threshold for collapse/readout DT = 0.05 # Time step for chunked evolution SIM_TIME = 3.0 # Total simulation time TEMPERATURE = 0.5 # Thermal bath temperature (in units where ħω/k_B = 1) DISORDER_STRENGTH = 0.3 # Std-dev of random local field disorder OMEGA_0 = 1.0 # Characteristic qubit frequency (sets n-bar scale) # ========================== QUBIT ========================== class LatticeQubit: """Single qubit with dipole-like local Hamiltonian, supporting disorder.""" def __init__(self, index: int, disorder_strength: float = 0.0, rng: Optional[np.random.Generator] = None): self.index = index rng = rng or np.random.default_rng() # Disorder: random shift drawn from N(0, disorder_strength) self.disorder_field = rng.normal(0.0, disorder_strength) if disorder_strength > 0 else 0.0 self.state = qt.basis(2, 0) # Start in |0⟩ def local_hamiltonian(self, coupling_strength: float = LOCAL_COUPLING) -> qt.Qobj: """ Local dipole energy + small tunnelling term + disorder field. H_local = coupling * (0.5 * σz + 0.1 * σx) + disorder * σz """ base = coupling_strength * (0.5 * qt.sigmaz() + 0.1 * qt.sigmax()) disorder_term = self.disorder_field * qt.sigmaz() return base + disorder_term # ========================== PROCESSOR ========================== class QuantumLatticeProcessor: """ 1D lattice quantum processor with coupled qubits. Supports: - Coherent XY evolution (corrected Hamiltonian) - Thermal Lindblad noise (temperature-dependent rates) - Disorder in local fields - Collective (superradiant) damping channel - Threshold collapse on energy variance """ def __init__( self, n_qubits: int = N_QUBITS, disorder_strength: float = DISORDER_STRENGTH, temperature: float = TEMPERATURE, collective_bath: bool = False, seed: int = 42, ): self.n = n_qubits self.temperature = temperature self.collective_bath = collective_bath rng = np.random.default_rng(seed) self.qubits = [LatticeQubit(i, disorder_strength, rng) for i in range(n_qubits)] self.hamiltonian = self._build_hamiltonian() self.current_state = qt.tensor([q.state for q in self.qubits]) # History tracking self.time_series: List[float] = [] self.coherence_history: List[float] = [] self.entanglement_history: List[float] = [] self.energy_var_history: List[float] = [] # ---- Hamiltonian ---- def _embed(self, op: qt.Qobj, site: int) -> qt.Qobj: """Embed a single-site operator into the full N-qubit space.""" ops = [qt.qeye(2)] * self.n ops[site] = op return qt.tensor(ops) def _build_hamiltonian(self) -> qt.Qobj: """ Build dipole + nearest-neighbour XY Hamiltonian. H = Σ_i H_local(i) + J Σ_i (σ+_i σ-_{i+1} + σ-_i σ+_{i+1}) V1 bug fixed: second term is σ-_i σ+_{i+1}, not σ+_i (σ-_{i+1})† = σ+_i σ+_{i+1}. """ H = 0 * qt.tensor([qt.qeye(2)] * self.n) # Local terms (include disorder via LatticeQubit.local_hamiltonian) for i, qubit in enumerate(self.qubits): H += self._embed(qubit.local_hamiltonian(), i) # Nearest-neighbour XY coupling (corrected) for i in range(self.n - 1): sp_i = self._embed(qt.sigmap(), i) sm_i = self._embed(qt.sigmam(), i) sp_j = self._embed(qt.sigmap(), i + 1) sm_j = self._embed(qt.sigmam(), i + 1) H += COUPLING_J * (sp_i * sm_j + sm_i * sp_j) return H # ---- Collapse operators ---- def _thermal_cops(self, rate: float = DECOHERENCE_RATE) -> List[qt.Qobj]: """ Temperature-dependent Lindblad operators. n_bar = 1 / (exp(ω/T) - 1) (sets occupation at temperature T) γ_down = rate * (n_bar + 1) (emission / decay) γ_up = rate * n_bar (absorption / excitation) """ n_bar = 1.0 / (np.exp(OMEGA_0 / max(self.temperature, 1e-9)) - 1.0) gamma_down = rate * (n_bar + 1.0) gamma_up = rate * n_bar c_ops = [] for i in range(self.n): c_ops.append(np.sqrt(gamma_down) * self._embed(qt.sigmam(), i)) if gamma_up > 1e-15: c_ops.append(np.sqrt(gamma_up) * self._embed(qt.sigmap(), i)) return c_ops def _collective_cops(self, rate: float = DECOHERENCE_RATE) -> List[qt.Qobj]: """ Collective (superradiant) Lindblad channel. Uses J- = Σ_i σ-_i (shared bath) instead of independent per-site operators. Collective decay rate scales as N*rate (Dicke superradiance scaling). """ J_minus = sum(self._embed(qt.sigmam(), i) for i in range(self.n)) J_plus = J_minus.dag() n_bar = 1.0 / (np.exp(OMEGA_0 / max(self.temperature, 1e-9)) - 1.0) gamma_down = rate * (n_bar + 1.0) gamma_up = rate * n_bar c_ops = [np.sqrt(gamma_down) * J_minus] if gamma_up > 1e-15: c_ops.append(np.sqrt(gamma_up) * J_plus) return c_ops # ---- Evolution ---- def evolve(self, t_list: np.ndarray, noisy: bool = False) -> None: """ Coherent (or noisy) evolution over t_list. Args: t_list: Time points for this chunk. noisy: If True, include thermal/collective Lindblad operators. """ if noisy: c_ops = (self._collective_cops() if self.collective_bath else self._thermal_cops()) else: c_ops = [] result = qt.mesolve(self.hamiltonian, self.current_state, t_list, c_ops=c_ops) self.current_state = result.states[-1] self.time_series.append(t_list[-1]) self.coherence_history.append(self._measure_coherence()) self.entanglement_history.append(self._measure_entanglement()) self.energy_var_history.append(self._energy_variance()) # ---- Metrics ---- def _density_matrix(self) -> qt.Qobj: """Return current density matrix (handles both pure and mixed states).""" if self.current_state.type == 'ket': return self.current_state * self.current_state.dag() return self.current_state def _measure_coherence(self) -> float: """ l1-norm of coherence: sum of absolute values of all off-diagonal elements. Replaces V1's single [0, -1] element, which was near-zero for physical states. """ rho = self._density_matrix().full() mask = ~np.eye(rho.shape[0], dtype=bool) return float(np.sum(np.abs(rho[mask]))) def _measure_entanglement(self) -> float: """Bipartite entanglement entropy (von Neumann, half-chain cut).""" rho = self._density_matrix() half = self.n // 2 rho_A = qt.ptrace(rho, list(range(half))) evals = rho_A.eigenenergies() evals = evals[evals > 1e-12] return float(-np.sum(evals * np.log2(evals + 1e-12))) def _energy_variance(self) -> float: """ Variance of energy: ⟨H²⟩ - ⟨H⟩². Used as the threshold trigger (replaces V1's raw ⟨H⟩, which almost always exceeded the threshold given J=2 coupling). """ E = qt.expect(self.hamiltonian, self.current_state) E2 = qt.expect(self.hamiltonian * self.hamiltonian, self.current_state) return float(E2 - E**2) def qubit_populations(self) -> np.ndarray: """ Per-qubit probability of being in |1⟩, computed via partial trace. V1 bug fixed: was indexing global amplitude 2^i instead of tracing out all other qubits. """ rho = self._density_matrix() pops = [] for i in range(self.n): rho_i = qt.ptrace(rho, [i]) pops.append(float(rho_i[1, 1].real)) return np.array(pops) # ---- Threshold collapse ---- def threshold_collapse(self) -> Dict: """ Projective measurement when energy variance exceeds threshold. V1 used raw ⟨H⟩ which fired almost every step. Energy variance is a physically meaningful trigger: it peaks when the state is a superposition of widely separated energy eigenstates. """ var = self._energy_variance() if var > COLLAPSE_THRESHOLD: rho = self._density_matrix() probs = np.real(np.diag(rho.full())) probs = np.clip(probs, 0, None) probs /= probs.sum() outcome_idx = np.random.choice(range(2**self.n), p=probs) self.current_state = qt.basis(2**self.n, outcome_idx) bitstring = bin(outcome_idx)[2:].zfill(self.n) return {"collapsed": True, "outcome": bitstring, "energy_var": var} return {"collapsed": False, "outcome": None, "energy_var": var} # ---- Gate fidelity ---- def apply_rabi_pulse(self, target_qubit: int, angle: float = np.pi) -> float: """ Apply a Rabi rotation pulse to a target qubit and return gate fidelity. Implements a simple Rx(angle) control pulse via mesolve with a drive term. Fidelity is computed against the ideal unitary applied to the initial state. Args: target_qubit: Index of qubit to pulse. angle: Rotation angle in radians (default π = NOT gate). Returns: Fidelity between evolved state and ideal target state. """ drive_strength = angle / (2 * np.pi) # Scale drive to achieve desired rotation H_drive = drive_strength * self._embed(qt.sigmax(), target_qubit) H_total = [self.hamiltonian, [H_drive, lambda t, args: np.cos(2 * np.pi * t)]] t_pulse = np.linspace(0, 1.0, 50) result = qt.mesolve(H_total, self.current_state, t_pulse, c_ops=[]) evolved = result.states[-1] # Ideal target: Rx(angle) on the target qubit Rx = (-1j * angle / 2 * qt.sigmax()).expm() ideal_op = qt.tensor([Rx if i == target_qubit else qt.qeye(2) for i in range(self.n)]) ideal_state = ideal_op * self.current_state fidelity = float(abs((ideal_state.dag() * evolved)[0, 0]) ** 2) return fidelity # ========================== JORDAN-WIGNER ANALYTICS ========================== class JordanWignerAnalytics: """ Free-fermion (Jordan-Wigner) analytics for the XY chain. The 1D XY model maps exactly to free fermions via: c_i = (∏_{j<i} σz_j) σ-_i giving H = Σ_k ε_k (d†_k d_k - 1/2) with ε_k = 2J cos(k) + local terms. Used as a correctness benchmark against the QuTiP simulation. """ def __init__(self, n: int, J: float = COUPLING_J, local_coupling: float = LOCAL_COUPLING): self.n = n self.J = J self.local_coupling = local_coupling def dispersion(self) -> np.ndarray: """Quasi-particle energies ε_k for the clean (no-disorder) chain.""" k_vals = np.pi * np.arange(1, self.n + 1) / (self.n + 1) local_e = self.local_coupling * 0.5 # σz contribution return 2 * self.J * np.cos(k_vals) + local_e def ground_state_energy(self) -> float: """Analytic ground state energy (sum over negative-energy modes).""" eps = self.dispersion() return float(np.sum(eps[eps < 0])) def build_hopping_matrix(self) -> np.ndarray: """Build the N×N hopping matrix for the free-fermion Hamiltonian.""" H_mat = np.zeros((self.n, self.n)) local_e = self.local_coupling * 0.5 for i in range(self.n): H_mat[i, i] = local_e for i in range(self.n - 1): H_mat[i, i + 1] = self.J H_mat[i + 1, i] = self.J return H_mat def analytic_energies(self) -> np.ndarray: """All single-particle energies from exact diagonalisation of hopping matrix.""" H_mat = self.build_hopping_matrix() evals, _ = eigh(H_mat) return evals def compare_with_qutip(self, processor: QuantumLatticeProcessor) -> Dict: """ Compare QuTiP ground state energy with Jordan-Wigner analytic result. Returns dict with both values and relative error. """ # QuTiP: lowest eigenvalue of full Hamiltonian qutip_evals = processor.hamiltonian.eigenenergies() qutip_gs = float(np.min(qutip_evals)) # Analytic JW (sum occupied modes in ground state) sp_evals = self.analytic_energies() jw_gs = float(np.sum(sp_evals[sp_evals < 0])) rel_error = abs(qutip_gs - jw_gs) / (abs(jw_gs) + 1e-15) return { "qutip_ground_state_energy": qutip_gs, "jordan_wigner_ground_energy": jw_gs, "relative_error": rel_error, "single_particle_energies": sp_evals, } # ========================== VISUALIZATION ========================== class LatticeVisualizer: """Plotting utilities for processor metrics and lattice state.""" def __init__(self, processor: QuantumLatticeProcessor): self.processor = processor def plot_metrics(self, save_path: str = "lattice_processor_metrics_v2.png"): """Plot coherence, entanglement entropy, and energy variance over time.""" fig, axs = plt.subplots(3, 1, figsize=(10, 10)) t = self.processor.time_series axs[0].plot(t, self.processor.coherence_history, 'g-', label='l1 Coherence') axs[0].set_ylabel('Coherence (l1-norm)') axs[0].legend(); axs[0].grid(True) axs[1].plot(t, self.processor.entanglement_history, color='orange', label='Entanglement Entropy') axs[1].set_ylabel('von Neumann Entropy (bits)') axs[1].legend(); axs[1].grid(True) axs[2].plot(t, self.processor.energy_var_history, 'b-', label='Energy Variance') axs[2].axhline(COLLAPSE_THRESHOLD, color='red', linestyle='--', label=f'Collapse threshold ({COLLAPSE_THRESHOLD})') axs[2].set_xlabel('Time'); axs[2].set_ylabel('Var(E)') axs[2].legend(); axs[2].grid(True) plt.suptitle('Quantum Lattice Processor Metrics (V2)') plt.tight_layout() plt.savefig(save_path, dpi=200) print(f"✅ Metrics plot saved as '{save_path}'") plt.show() def plot_lattice_state(self, ax: plt.Axes, title: str): """ Visualize per-qubit |1⟩ populations. V1 bug fixed: now uses ptrace (via qubit_populations()) instead of directly indexing global amplitude 2^i. """ ax.set_title(title) pops = self.processor.qubit_populations() ax.set_xlim(0, self.processor.n + 1) ax.set_ylim(0, 3) for i, p1 in enumerate(pops): color = plt.cm.RdYlGn(1.0 - p1) ax.add_patch(plt.Rectangle((i + 0.2, 1), 0.6, 1.5, color=color, alpha=0.85)) ax.text(i + 0.5, 0.6, f'Q{i}', ha='center', fontsize=9) ax.text(i + 0.5, 2.7, f'{p1:.2f}', ha='center', fontsize=8) ax.set_yticks([]) sm = plt.cm.ScalarMappable(cmap='RdYlGn', norm=plt.Normalize(0, 1)) plt.colorbar(sm, ax=ax, label='P(|1⟩)', orientation='horizontal', pad=0.2) def plot_dispersion(self, jw: JordanWignerAnalytics): """Plot Jordan-Wigner single-particle dispersion relation.""" eps = jw.analytic_energies() plt.figure(figsize=(7, 4)) plt.plot(range(1, jw.n + 1), eps, 'o-', color='purple') plt.axhline(0, color='k', linestyle='--', linewidth=0.8) plt.xlabel('Mode index k') plt.ylabel('Energy ε_k') plt.title('Jordan-Wigner Single-Particle Dispersion') plt.grid(True) plt.tight_layout() plt.savefig("jw_dispersion.png", dpi=200) print("✅ Dispersion plot saved as 'jw_dispersion.png'") plt.show() # ========================== MAIN ENGINE ========================== def run_quantum_processor_experiment(): print("=" * 60) print(" MICROQUANT V2 — Quantum Lattice Processor Simulator") print("=" * 60) # --- Setup --- print(f"\n🔧 Configuration:") print(f" N qubits: {N_QUBITS}") print(f" Coupling J: {COUPLING_J}") print(f" Disorder strength: {DISORDER_STRENGTH}") print(f" Temperature: {TEMPERATURE}") print(f" Collective bath: False (independent Lindblad)") processor = QuantumLatticeProcessor( n_qubits=N_QUBITS, disorder_strength=DISORDER_STRENGTH, temperature=TEMPERATURE, collective_bath=False, seed=42, ) # --- Jordan-Wigner benchmark (correctness check) --- print("\n📐 Jordan-Wigner Analytics (correctness benchmark):") jw = JordanWignerAnalytics(N_QUBITS) comparison = jw.compare_with_qutip(processor) print(f" QuTiP ground state energy: {comparison['qutip_ground_state_energy']:.6f}") print(f" Jordan-Wigner analytic: {comparison['jordan_wigner_ground_energy']:.6f}") print(f" Relative error: {comparison['relative_error']:.2e}") if comparison['relative_error'] < 0.01: print(" ✅ Simulation matches analytics (error < 1%)") else: print(" ⚠️ Mismatch — check Hamiltonian parameters") # --- Gate fidelity test --- print("\n🎯 Gate Fidelity Test (Rabi π-pulse on qubit 0):") fidelity = processor.apply_rabi_pulse(target_qubit=0, angle=np.pi) print(f" Rabi π-pulse fidelity: {fidelity:.4f}") # --- Coherent + noisy evolution --- print("\n🚀 Running quantum evolution...") t_steps = np.arange(0, SIM_TIME, DT) n_chunks = len(t_steps) // 10 for step in range(n_chunks): t_chunk = t_steps[step * 10:(step + 1) * 10] # Mix coherent and noisy evolution noisy = (step % 3 == 0) processor.evolve(t_chunk, noisy=noisy) # Threshold collapse check if step % 4 == 0: result = processor.threshold_collapse() if result["collapsed"]: print(f" → Collapse at t≈{t_chunk[-1]:.2f} | " f"Outcome: {result['outcome']} | Var(E)={result['energy_var']:.3f}") # --- Summary --- print(f"\n📊 FINAL RESULTS") print(f" Peak l1-coherence: {max(processor.coherence_history):.4f}") print(f" Final l1-coherence: {processor.coherence_history[-1]:.4f}") print(f" Max entanglement: {max(processor.entanglement_history):.4f} bits") print(f" Max energy variance: {max(processor.energy_var_history):.4f}") print(f" Collapse threshold: {COLLAPSE_THRESHOLD}") collapses = sum(1 for v in processor.energy_var_history if v > COLLAPSE_THRESHOLD) print(f" Steps above threshold: {collapses}/{len(processor.energy_var_history)}") # --- Disorder report --- print(f"\n🌀 Disorder Fields (per qubit):") for q in processor.qubits: print(f" Q{q.index}: δ = {q.disorder_field:+.4f}") # --- Visualization --- viz = LatticeVisualizer(processor) viz.plot_metrics() viz.plot_dispersion(jw) fig, ax = plt.subplots(figsize=(10, 3)) viz.plot_lattice_state(ax, "Final Qubit Populations") plt.tight_layout() plt.savefig("lattice_state_final.png", dpi=200) print("✅ Final lattice state saved as 'lattice_state_final.png'") plt.show() print("\n✅ MicroQuant V2 simulation complete.") print(" Input: |0⟩^N → XY Evolution + Thermal Bath → Threshold Readout") if __name__ == "__main__": run_quantum_processor_experiment()

Suggested Next Steps (V3)

Quantum optimal control (QOC): Use QuTiP's qutip_qoc package to optimize pulse shapes for multi-qubit gates and maximize fidelity under decoherence.

Long-range interactions: Extend XY coupling beyond nearest-neighbour (1/r³ for dipole-dipole), relevant to the microtubule biological framing.

Polariton model: Add a bosonic cavity mode (Jaynes-Cummings coupling) using qt.destroy(n_fock) — requires truncating Fock space.

Quantum volume metric: Apply random 2-qubit circuits and compute heavy output probability as a hardware-benchmarking analog.

Finite-size scaling: Sweep N from 4 to 12 and observe entanglement entropy scaling at the disorder-driven localization transition.

MicroQuant V5 — Microtubule Lattice Quantum Processor Simulator

Version 5.0 — Integrates all V5 improvements over V4:

Performance: JAX/Diffrax ODE backend for GPU-accelerated time evolution (replaces QuTiP mesolve CPU bottleneck); automatic fallback to QuTiP when JAX unavailable.

Lattice dimensionality: Full 2D cylindrical N × 13 protofilament grid with periodic angular boundaries; single-PF 1D chain retained as special case (n_protofilaments=1).

Measurement/feedback: Continuous homodyne measurement via stochastic master equation (smesolve); optional Bayesian feedback controller.

Open vs periodic boundaries: Proper minimum-image convention for all pair distances in long-range dipole sums on both 1D chain and 2D cylinder.

Validation: Benchmark suite against published microtubule quantum-coherence literature (Bandyopadhyay, Tuszynski groups); configurable comparison metrics.

""" physics_constants.py Microtubule-specific physical constants, unit conversion helpers, and 3D helical geometry for the full 13-protofilament cylinder. V5 additions ------------ - site_position_cylinder(): 2D (PF, site) → 3D Cartesian on the helix. - minimum_image_displacement(): proper periodic minimum-image vector for both 1D (axial only) and 2D (axial + angular) periodic boundaries. - All internal calculations remain in natural units (ħ = k_B = 1, energy in meV). Set biological_units=True in SimulationConfig to convert outputs. """ import math import numpy as np from typing import Tuple # ── Microtubule geometry ────────────────────────────────────────────────────── LATTICE_SPACING_NM = 8.0 # tubulin dimer axial spacing (nm) N_PROTOFILAMENTS = 13 # standard B-type microtubule OUTER_RADIUS_NM = 12.5 # outer cylinder radius (nm) HELIX_RISE_PER_DIMER = 0.923 # axial rise per dimer along the helix (nm) HELIX_TWIST_RAD = 2 * np.pi / N_PROTOFILAMENTS # Δφ between PFs # ── Dipole parameters ───────────────────────────────────────────────────────── TUBULIN_DIPOLE_DEBYE = 400.0 # tubulin electric dipole moment (Debye) DIPOLE_MOMENT_SI = TUBULIN_DIPOLE_DEBYE * 3.336e-30 # C·m # ── Energy / coupling scales ────────────────────────────────────────────────── COUPLING_J_MEV = 5e-3 # XY coupling J (meV) COUPLING_J = COUPLING_J_MEV CAVITY_FREQ_MEV = 1.0 # cavity mode frequency (meV) G_POLARITON_MEV = 0.5 # light-matter coupling (meV) # ── Decoherence / thermal ───────────────────────────────────────────────────── GAMMA_BASE = 0.01 # base amplitude damping rate (meV) BIOLOGICAL_TEMP_K = 310.0 # physiological temperature (K) K_B_MEV_PER_K = 8.617e-2 # Boltzmann constant (meV / K) # ── Unit conversions ────────────────────────────────────────────────────────── MEV_TO_GHZ = 241.8 # 1 meV ≈ 241.8 GHz MEV_TO_WAVENUMBER = 8.066 # 1 meV ≈ 8.066 cm⁻¹ # ── Thermal occupation ──────────────────────────────────────────────────────── def n_thermal(omega_mev: float, T_K: float = BIOLOGICAL_TEMP_K) -> float: """Bose–Einstein occupation number at frequency omega_mev and temperature T_K.""" kT = K_B_MEV_PER_K * T_K if kT < 1e-12: return 0.0 x = omega_mev / kT return 1.0 / (math.exp(x) - 1.0) if x > 1e-6 else kT / omega_mev def mev_to_kelvin(e_mev: float) -> float: """Convert energy in meV to equivalent temperature in Kelvin.""" return e_mev / K_B_MEV_PER_K # ── 3D Helical geometry ─────────────────────────────────────────────────────── def site_position_3d( site : int, protofilament : int = 0, radius_nm : float = OUTER_RADIUS_NM, spacing_nm : float = LATTICE_SPACING_NM, ) -> np.ndarray: """ Return the 3D Cartesian position (nm) of a tubulin dimer on the helix. Parameters ---------- site : Index along the protofilament axis. protofilament : Angular index 0 … N_PROTOFILAMENTS-1. radius_nm : Cylinder radius. spacing_nm : Axial spacing between dimers along one protofilament. Returns ------- np.ndarray of shape (3,) — [x, y, z] in nm. """ phi = protofilament * HELIX_TWIST_RAD x = radius_nm * np.cos(phi) y = radius_nm * np.sin(phi) z = site * spacing_nm + protofilament * HELIX_RISE_PER_DIMER return np.array([x, y, z]) # Alias used by CylindricalLattice site_position_cylinder = site_position_3d def dipole_unit_vector(theta: float, phi: float) -> np.ndarray: """ 3D unit vector for a dipole with polar angle θ and azimuthal angle φ. """ return np.array([ np.sin(theta) * np.cos(phi), np.sin(theta) * np.sin(phi), np.cos(theta), ]) def angular_factor_3d( p_i : np.ndarray, p_j : np.ndarray, r_vec : np.ndarray, ) -> float: """ Full dipole–dipole angular prefactor: A = (p̂_i · p̂_j) − 3 (r̂ · p̂_i)(r̂ · p̂_j) Parameters ---------- p_i, p_j : Unit vectors of dipole orientations (shape (3,)). r_vec : Displacement vector i→j (any units). Returns ------- float — Angular factor (dimensionless). """ r_norm = np.linalg.norm(r_vec) if r_norm < 1e-12: return 0.0 r_hat = r_vec / r_norm return float(np.dot(p_i, p_j) - 3.0 * np.dot(r_hat, p_i) * np.dot(r_hat, p_j)) # ── Minimum-image convention ────────────────────────────────────────────────── def minimum_image_displacement( r_i : np.ndarray, r_j : np.ndarray, box_lengths : np.ndarray, periodic : np.ndarray, ) -> np.ndarray: """ Compute the minimum-image displacement vector r_j − r_i under periodic boundary conditions. V5 addition: replaces the naïve (j − i) lattice-index distance used in V4's periodic flag, which was only correct for the nearest-neighbour wrap-around bond and gave wrong distances for all longer-range pairs. Parameters ---------- r_i, r_j : 3D Cartesian positions (nm). box_lengths : Box dimensions in each Cartesian direction (nm). Use 0 or np.inf for non-periodic directions. periodic : Boolean array of length 3 — which directions are periodic. Returns ------- np.ndarray of shape (3,) — minimum-image displacement (nm). Notes ----- For the 2D cylindrical lattice the x-y plane is curved; this function operates on the unrolled Cartesian coordinates. For the angular direction of the cylinder, CylindricalLattice converts the arc-length periodicity to an effective box length 2π R before calling here. """ dr = r_j - r_i for k in range(3): if periodic[k] and box_lengths[k] > 1e-12: dr[k] -= box_lengths[k] * np.round(dr[k] / box_lengths[k]) return dr def minimum_image_distance( r_i : np.ndarray, r_j : np.ndarray, box_lengths : np.ndarray, periodic : np.ndarray, ) -> float: """Scalar Euclidean distance under the minimum-image convention.""" dr = minimum_image_displacement(r_i, r_j, box_lengths, periodic) return float(np.linalg.norm(dr))

core/jax_integrator.py

""" jax_integrator.py JAX/Diffrax GPU-accelerated Lindblad master equation integrator. V5 replaces QuTiP's CPU-bound mesolve with a vectorised ODE solver that runs on GPU (or CPU if no GPU is present). The interface mirrors QuTiP's mesolve output so that QuantumLatticeProcessor can swap backends transparently. Design ------ - The density matrix ρ (d×d complex) is reshaped to a flat real vector of length 2d² (interleaved real/imag) for Diffrax's ODE solver. - The Lindblad RHS is compiled once with jax.jit and reused across all time steps. - Gradient of the fidelity w.r.t. Hamiltonian parameters is available via jax.grad for future variational applications. Dependencies ------------ pip install jax jaxlib diffrax Falls back gracefully when JAX is unavailable: import this module and call integrate(); if JAX is not installed it raises ImportError with a clear message pointing to the QuTiP fallback. References ---------- Lindblad equation: dρ/dt = −i[H, ρ] + Σ_k (L_k ρ L_k† − ½{L_k†L_k, ρ}) """ from __future__ import annotations import numpy as np from typing import List, Optional _JAX_AVAILABLE = False try: import jax import jax.numpy as jnp import diffrax _JAX_AVAILABLE = True except ImportError: pass def _check_jax() -> None: if not _JAX_AVAILABLE: raise ImportError( "JAX and/or Diffrax not installed. Install with:\n" " pip install jax jaxlib diffrax\n" "Or set SimulationConfig.use_jax=False to use the QuTiP backend." ) # ── Internal helpers ────────────────────────────────────────────────────────── def _rho_to_vec(rho: np.ndarray) -> np.ndarray: """Flatten complex d×d → real 2d² (real part first, then imag).""" d2 = rho.size return np.concatenate([rho.real.ravel(), rho.imag.ravel()]) def _vec_to_rho(v: np.ndarray, d: int) -> np.ndarray: """Reconstruct complex d×d from flat real 2d² vector.""" return v[:d*d].reshape(d, d) + 1j * v[d*d:].reshape(d, d) def _build_lindblad_rhs_jax(H: np.ndarray, L_ops: List[np.ndarray]): """ Return a JAX-jit-compiled Lindblad RHS function. Parameters ---------- H : Hamiltonian as complex numpy array (d×d). L_ops : List of Lindblad operators (each d×d complex numpy array). Returns ------- Callable (t, y, args) → dy/dt (Diffrax convention) """ _check_jax() d = H.shape[0] H_jax = jnp.array(H) Ls = [jnp.array(L) for L in L_ops] # Precompute L†L for each operator LdL = [L.conj().T @ L for L in Ls] @jax.jit def rhs(t, y, args): rho_r = y[:d*d].reshape(d, d) rho_i = y[d*d:].reshape(d, d) rho = rho_r + 1j * rho_i # Coherent part: −i[H, ρ] comm = H_jax @ rho - rho @ H_jax drho = -1j * comm # Dissipator: Σ_k (L_k ρ L_k† − ½{L_k†L_k, ρ}) for L, LdLk in zip(Ls, LdL): drho = drho + ( L @ rho @ L.conj().T - 0.5 * (LdLk @ rho + rho @ LdLk) ) return jnp.concatenate([drho.real.ravel(), drho.imag.ravel()]) return rhs, d class JaxLindbladResult: """Minimal result object matching the fields QuantumLatticeProcessor uses.""" def __init__(self, states: List[np.ndarray]): self.states = states # list of qutip-compatible density matrices def integrate( H : "qt.Qobj", # QuTiP Qobj (converted internally) rho0 : "qt.Qobj", tlist : np.ndarray, c_ops : List["qt.Qobj"], *, rtol : float = 1e-6, atol : float = 1e-8, ) -> JaxLindbladResult: """ GPU-accelerated Lindblad integration via JAX/Diffrax. Parameters ---------- H, rho0, c_ops : QuTiP objects (converted to dense numpy internally). tlist : 1D array of output times. rtol, atol : Solver tolerances. Returns ------- JaxLindbladResult with `.states` list of QuTiP Qobj density matrices. """ _check_jax() import qutip as qt H_np = H.full() r0_np = rho0.full() if rho0.type == "oper" else qt.ket2dm(rho0).full() Ls_np = [L.full() for L in c_ops] d = H_np.shape[0] rhs, _ = _build_lindblad_rhs_jax(H_np, Ls_np) y0 = jnp.array(_rho_to_vec(r0_np)) t0, t1 = float(tlist[0]), float(tlist[-1]) term = diffrax.ODETerm(rhs) solver = diffrax.Dopri5() saveat = diffrax.SaveAt(ts=jnp.array(tlist, dtype=jnp.float64)) stepctl = diffrax.PIDController(rtol=rtol, atol=atol) sol = diffrax.diffeqsolve( term, solver, t0=t0, t1=t1, dt0=(tlist[1] - tlist[0]) * 0.1, y0=y0, saveat=saveat, stepsize_controller=stepctl, max_steps=int(1e6), ) # Convert output back to QuTiP Qobj list states = [] dims = [[2] * int(round(np.log2(d))), [2] * int(round(np.log2(d)))] for k in range(len(tlist)): rho_k = _vec_to_rho(np.array(sol.ys[k]), d) # Enforce Hermiticity (numerical drift) rho_k = 0.5 * (rho_k + rho_k.conj().T) qobj = qt.Qobj(rho_k, dims=dims) states.append(qobj) return JaxLindbladResult(states)

""" CylindricalLattice.py Full 2D N × 13-protofilament cylindrical lattice builder. V5 new module ------------- Builds the complete site list and coupling graph for a 2D microtubule cylinder. Replaces the implicit single-protofilament (pf=0) assumption in V4's QuantumLatticeProcessor._init_qubits(). Layout ------ Site index: flat_index = pf * N + site (pf ∈ [0, n_pf), site ∈ [0, N)) Total sites: n_total = N * n_pf Periodicity ----------- - Axial (z): optional, controlled by config.periodic. - Angular (φ): optional, controlled by config.periodic_angular. Angular wrap: PF 12 → PF 0 (true cylinder topology). Box lengths for minimum-image ------------------------------- - Axial box length = N * LATTICE_SPACING_NM - Angular box length = 2π * OUTER_RADIUS_NM (arc-length of full circle) - z box for x and y: np.inf (not periodic in Cartesian x/y) """ import numpy as np from typing import List, Tuple, Optional import qutip as qt from core.LatticeQubit import LatticeQubit from core.physics_constants import ( site_position_cylinder, minimum_image_displacement, OUTER_RADIUS_NM, LATTICE_SPACING_NM, N_PROTOFILAMENTS, ) class CylindricalLattice: """ Build the full qubit list and adjacency for an N × n_pf cylinder. Parameters ---------- N : int Number of sites per protofilament (axial). n_pf : int Number of protofilaments (default 13). periodic_axial : bool Periodic boundary in z direction. periodic_angular: bool Periodic boundary in φ direction. disorder_fields : np.ndarray of shape (N * n_pf,) or None. rng : np.random.Generator for angle randomisation. """ def __init__( self, N : int, n_pf : int = N_PROTOFILAMENTS, periodic_axial : bool = False, periodic_angular : bool = False, disorder_fields : Optional[np.ndarray] = None, rng : Optional[np.random.Generator] = None, ): self.N = N self.n_pf = n_pf self.n_total = N * n_pf self.periodic_axial = periodic_axial self.periodic_ang = periodic_angular self.rng = rng or np.random.default_rng(42) if disorder_fields is None: disorder_fields = np.zeros(self.n_total) assert len(disorder_fields) == self.n_total self._thetas = self.rng.uniform(0, np.pi, self.n_total) self._phis = self.rng.uniform(0, 2 * np.pi, self.n_total) self.qubits: List[LatticeQubit] = self._build_qubits(disorder_fields) # Box lengths for minimum-image convention self._box = np.array([ np.inf, # x (radial — not periodic) np.inf, # y (radial — not periodic) N * LATTICE_SPACING_NM, # z (axial) ]) # Angular periodicity handled via arc-length in the z-component remapping # (see _min_image_r_vec for details). # ── Flat index helpers ──────────────────────────────────────────────────── def flat_index(self, pf: int, site: int) -> int: return pf * self.N + site def pf_site(self, flat: int) -> Tuple[int, int]: return divmod(flat, self.N) # ── Qubit construction ──────────────────────────────────────────────────── def _build_qubits(self, disorder_fields: np.ndarray) -> List[LatticeQubit]: qubits = [] for pf in range(self.n_pf): for site in range(self.N): idx = self.flat_index(pf, site) qubits.append(LatticeQubit( site = site, protofilament = pf, energy_gap = 1.0, disorder_field = disorder_fields[idx], dipole_theta = self._thetas[idx], dipole_phi = self._phis[idx], )) return qubits # ── Minimum-image displacement on cylinder ──────────────────────────────── def _min_image_r_vec(self, idx_i: int, idx_j: int) -> np.ndarray: """ Minimum-image displacement between sites idx_i and idx_j. Axial (z) periodic images are handled via the minimum-image convention on the axial box length. Angular periodic images: the angular coordinate is discrete (n_pf slots). When periodic_angular=True we choose the PF offset Δpf ∈ (−n_pf/2, n_pf/2] before computing Cartesian positions, ensuring we always use the shorter arc. """ pf_i, site_i = self.pf_site(idx_i) pf_j, site_j = self.pf_site(idx_j) # Angular minimum image if self.periodic_ang: dpf = pf_j - pf_i # Wrap to (−n_pf/2, n_pf/2] if dpf > self.n_pf // 2: dpf -= self.n_pf elif dpf < -(self.n_pf // 2): dpf += self.n_pf pf_j_eff = pf_i + dpf else: pf_j_eff = pf_j r_i = site_position_cylinder(site_i, pf_i) r_j = site_position_cylinder(site_j, pf_j_eff) # Axial minimum image periodic_flags = np.array([False, False, self.periodic_axial]) return minimum_image_displacement(r_i, r_j, self._box, periodic_flags) def displacement_matrix(self) -> np.ndarray: """ Return the (n_total × n_total × 3) array of minimum-image displacement vectors. Cached after first call. """ if hasattr(self, "_disp_cache"): return self._disp_cache n = self.n_total D = np.zeros((n, n, 3)) for i in range(n): for j in range(n): if i != j: D[i, j] = self._min_image_r_vec(i, j) self._disp_cache = D return D def distance_matrix(self) -> np.ndarray: """Scalar minimum-image distance matrix (n_total × n_total).""" D = self.displacement_matrix() return np.linalg.norm(D, axis=-1) # ── Nearest-neighbour adjacency ─────────────────────────────────────────── def nn_pairs(self) -> List[Tuple[int, int]]: """ Return list of (i, j) nearest-neighbour bonds: - axial: (pf, site) ↔ (pf, site+1) - angular: (pf, site) ↔ (pf+1, site) Respects periodic flags. """ pairs = [] for pf in range(self.n_pf): for site in range(self.N): i = self.flat_index(pf, site) # Axial bond if site < self.N - 1: pairs.append((i, self.flat_index(pf, site + 1))) elif self.periodic_axial: pairs.append((i, self.flat_index(pf, 0))) # Angular bond if pf < self.n_pf - 1: pairs.append((i, self.flat_index(pf + 1, site))) elif self.periodic_ang: pairs.append((i, self.flat_index(0, site))) return pairs

measurement/HomodyneSME.py

""" HomodyneSME.py Continuous homodyne measurement via the stochastic master equation. V5 new module ------------- Implements the homodyne (diffusive) unravelling of the Lindblad master equation: dρ = (-i[H, ρ] + D[L]ρ) dt + sqrt(η) (L ρ + ρ L† − Tr[(L + L†)ρ] ρ) dW where η ∈ (0,1] is the detector efficiency and dW is an Itô Wiener increment. Two backends ------------ 1. QuTiP smesolve — used when use_jax=False (default). 2. Custom Euler-Milstein JAX loop — used when use_jax=True (faster on GPU for large trajectories, but less adaptive step-size control). The photocurrent record is also returned so BayesianFeedback can consume it. References ---------- Wiseman & Milburn (2010), "Quantum Measurement and Control", Chapter 4. Jacobs & Steck (2006), Contemp. Phys. 47, 279. """ from __future__ import annotations import warnings import numpy as np from typing import List, Optional, Tuple, TYPE_CHECKING if TYPE_CHECKING: import qutip as qt class HomodyneResult: """Container for stochastic evolution results.""" def __init__( self, states : List["qt.Qobj"], times : np.ndarray, photocurrent : np.ndarray, trajectories : Optional[List[List["qt.Qobj"]]] = None, ): self.states = states # ensemble-averaged state at each t self.times = times self.photocurrent = photocurrent # shape (n_traj, n_steps) self.trajectories = trajectories # individual trajectories if kept class HomodyneSME: """ Stochastic master equation under homodyne detection. Parameters ---------- H : qt.Qobj Drift Hamiltonian. c_ops : list Lindblad operators (environment channels). meas_op : qt.Qobj Measurement operator L (defines homodyne channel). efficiency : float Detector efficiency η. n_traj : int Number of stochastic trajectories. seed : int RNG seed for reproducibility. use_jax : bool Use JAX Euler-Milstein if True. """ try: import jax.numpy as jnp import jax except ImportError: warnings.warn( "JAX not available; falling back to QuTiP smesolve.", UserWarning ) return self._run_qutip(rho0, tlist) import qutip as qt H_np = self.H.full() L_np = self.L.full() c_nps = [c.full() for c in self.c_ops] rho0_np = rho0.full() if rho0.type == "oper" else qt.ket2dm(rho0).full() d = H_np.shape[0] dt = float(tlist[1] - tlist[0]) @jax.jit def step(rho, xi): # Lindblad drift comm = H_np @ rho - rho @ H_np drho = -1j * comm for Lk in c_nps: drho += Lk @ rho @ Lk.conj().T - 0.5 * (Lk.conj().T @ Lk @ rho + rho @ Lk.conj().T @ Lk) # Homodyne innovation G = L_np @ rho + rho @ L_np.conj().T tr_G = jnp.trace(G).real innov = G - tr_G * rho rho_new = rho + dt * drho + jnp.sqrt(self.eta * dt) * innov * xi # Renormalise rho_new = 0.5 * (rho_new + rho_new.conj().T) rho_new = rho_new / jnp.trace(rho_new).real return rho_new H_np = jnp.array(H_np) L_np = jnp.array(L_np) c_nps = [jnp.array(c) for c in c_nps] all_states = [] all_pc = np.zeros((self.n_traj, len(tlist))) for tr in range(self.n_traj): rho = jnp.array(rho0_np) traj = [] noises = self.rng.standard_normal(len(tlist)) for k, _ in enumerate(tlist): xi = float(noises[k]) rho = step(rho, xi) traj.append(np.array(rho)) # Photocurrent I = sqrt(η) <L + L†> + ξ/sqrt(dt) rho_np = np.array(rho) all_pc[tr, k] = ( np.sqrt(self.eta) * np.real(np.trace((L_np + L_np.conj().T) @ rho_np)) + xi / np.sqrt(dt) ) all_states.append(traj) dims = [[2] * int(round(np.log2(d))), [2] * int(round(np.log2(d)))] avg_states_np = [ sum(all_states[tr][k] for tr in range(self.n_traj)) / self.n_traj for k in range(len(tlist)) ] avg_qobj = [qt.Qobj(s, dims=dims) for s in avg_states_np] return HomodyneResult( states = avg_qobj, times = tlist, photocurrent = all_pc, )

""" BayesianFeedback.py Real-time Bayesian state estimator for continuous homodyne measurement. V5 new module ------------- Given the photocurrent record {I(t_k)} from HomodyneSME, this module maintains a running estimate of the quantum state ρ̂(t) using the stochastic master equation in its Kushner-Stratonovich form. Feedback -------- Optional closed-loop feedback: at each step, a feedback Hamiltonian H_fb(ρ̂) is computed from the estimated state and applied to the system. Two built-in feedback laws are provided: - 'wiseman_milburn' : H_fb = −ℏ θ(t) σy, where θ(t) is chosen to maximise the purity of ρ̂. - 'lqg' : Linear-quadratic-Gaussian (proportional control on the Bloch-sphere z-component). References ---------- Wiseman & Milburn (2010), Chapter 5. Jacobs (2014), "Quantum Measurement Theory and its Applications", Ch. 6. """ import numpy as np from typing import Callable, List, Optional, Tuple import qutip as qt class BayesianFeedback: """ Online Bayesian state estimator with optional closed-loop feedback. Parameters ---------- H_drift : qt.Qobj Drift Hamiltonian. L_meas : qt.Qobj Measurement operator. c_ops : list Additional Lindblad operators. efficiency : float Detector efficiency η. feedback_law: str 'none' | 'wiseman_milburn' | 'lqg'. feedback_gain: float Proportional gain for feedback laws. n_qubits : int Number of qubits (for operator construction). """ def __init__( self, H_drift : qt.Qobj, L_meas : qt.Qobj, c_ops : List[qt.Qobj], efficiency : float = 0.9, feedback_law : str = "none", feedback_gain: float = 1.0, n_qubits : int = 1, ): self.H0 = H_drift.full() self.L = L_meas.full() self.c_ops = [c.full() for c in c_ops] self.eta = efficiency self.law = feedback_law self.gain = feedback_gain self.n_q = n_qubits self.d = H_drift.shape[0] # Build feedback-relevant operators self._Sy = sum( qt.tensor([qt.qeye(2)] * i + [qt.sigmay()] + [qt.qeye(2)] * (n_qubits - i - 1)).full() for i in range(n_qubits) ) self._Sz = sum( qt.tensor([qt.qeye(2)] * i + [qt.sigmaz()] + [qt.qeye(2)] * (n_qubits - i - 1)).full() for i in range(n_qubits) ) def _lindblad_step(self, rho: np.ndarray, dt: float) -> np.ndarray: """Deterministic Lindblad drift (no measurement).""" H_np = self.H0 comm = H_np @ rho - rho @ H_np drho = -1j * comm for Lk in self.c_ops: drho += Lk @ rho @ Lk.conj().T - 0.5 * (Lk.conj().T @ Lk @ rho + rho @ Lk.conj().T @ Lk) return rho + dt * drho def _measurement_update( self, rho: np.ndarray, I_t: float, dt: float ) -> np.ndarray: """ Bayesian update from homodyne photocurrent I(t). Stochastic update (Euler-Maruyama form of diffusive SME): dρ = ... dt + sqrt(η)(L ρ + ρ L† − Tr[(L+L†)ρ] ρ) dW with dW = (I(t) − sqrt(η) <L+L†>) dt. """ G = self.L @ rho + rho @ self.L.conj().T tr_G = np.real(np.trace(G)) innov = (I_t - np.sqrt(self.eta) * tr_G) * dt rho_new = rho + np.sqrt(self.eta) * (G - tr_G * rho) * innov return rho_new def _feedback_hamiltonian(self, rho: np.ndarray) -> np.ndarray: """Compute the feedback Hamiltonian from the current state estimate.""" if self.law == "none": return np.zeros_like(self.H0) elif self.law == "wiseman_milburn": # Feedback angle proportional to off-diagonal coherence magnitude theta = self.gain * np.abs(rho[0, 1]) return theta * self._Sy elif self.law == "lqg": # Proportional control on Bloch z-component sz_exp = np.real(np.trace(self._Sz @ rho)) return self.gain * sz_exp * self._Sz else: raise ValueError(f"Unknown feedback law: '{self.law}'") def run( self, rho0 : qt.Qobj, tlist : np.ndarray, photocurrent : np.ndarray, ) -> Tuple[List[qt.Qobj], np.ndarray]: """ Run the Bayesian state estimator on a photocurrent record. Parameters ---------- rho0 : Initial state estimate. tlist : Time array matching photocurrent columns. photocurrent : 1D array of I(t) values (single trajectory). Returns ------- states_est : List of QuTiP Qobj estimated states. purity_trace : Array of Tr(ρ̂²) — purity of the estimate. """ import qutip as qt rho = rho0.full() if rho0.type == "oper" else qt.ket2dm(rho0).full() dt = float(tlist[1] - tlist[0]) d = self.d dims = [[2] * self.n_q, [2] * self.n_q] states_est = [] purity_trace = np.zeros(len(tlist)) for k, t in enumerate(tlist): # 1. Lindblad drift rho = self._lindblad_step(rho, dt) # 2. Measurement update I_t = photocurrent[k] rho = self._measurement_update(rho, I_t, dt) # 3. Feedback (modifies effective Hamiltonian for next step) if self.law != "none": H_fb = self._feedback_hamiltonian(rho) self.H0 = self.H0 + H_fb * dt # additive update # 4. Normalise & enforce Hermiticity rho = 0.5 * (rho + rho.conj().T) rho = rho / np.real(np.trace(rho)) states_est.append(qt.Qobj(rho.copy(), dims=dims)) purity_trace[k] = float(np.real(np.trace(rho @ rho))) return states_est, purity_trace

── Displacement vector (minimum-image aware) ──────────────── if self._cyl is not None: r_vec = disp_mat[i, j] else: r_i = self.qubits[i].position_3d r_j = self.qubits[j].position_3d # 1D minimum image along z if self.cfg.periodic: box_z = self.n * 8.0 # Distance in units of lattice spacing (dimensionless) r_lattice = r / 8.0 # LATTICE_SPACING_NM = 8.0 J_ij = COUPLING_J * ang / (r_lattice ** 3) sp_i = self._embed(qt.sigmap(), i) sm_i = self._embed(qt.sigmam(), i) sp_j = self._embed(qt.sigmap(), j) sm_j = self._embed(qt.sigmam(), j) H += J_ij * (sp_i * sm_j + sm_i * sp_j) return H def _H_polariton(self) -> qt.Qobj: ω_c = self.cfg.cavity_freq g = self.cfg.g_polariton a_f = self._embed_cavity(self.a) ad_f = self._embed_cavity(self.adag) H_cav = ω_c * ad_f * a_f for i in range(self.n): sp_i = self._embed(qt.sigmap(), i) sm_i = self._embed(qt.sigmam(), i) H_cav+= g * (sp_i * a_f + sm_i * ad_f) return H_cav # ── Collapse operators ──────────────────────────────────────────────────── def _build_collapse_ops(self) -> List[qt.Qobj]: cfg = self.cfg nf = self.n_fock mode = cfg.noise if mode == NoiseType.NONE: return [] elif mode == NoiseType.THERMAL_UNIFORM: return self._cops_thermal_uniform() elif mode == NoiseType.PHONON_CORRELATED: return PhononBath(self.n, cfg.gamma_base, cfg.temperature_K, cfg.correlation_length, nf).collapse_operators() elif mode == NoiseType.DIELECTRIC_1F: return DielectricDephasing(self.n, cfg.dephasing_rate_1f, n_fock=nf).collapse_operators() elif mode == NoiseType.FULL_BIOLOGICAL: return FullBiologicalNoise(self.n, cfg, nf).collapse_operators() else: raise ValueError(f"Unknown noise type: {mode}") def _cops_thermal_uniform(self) -> List[qt.Qobj]: n_bar = n_thermal(1.0, self.cfg.temperature_K) γ_down = self.cfg.gamma_base * (n_bar + 1.0) γ_up = self.cfg.gamma_base * n_bar nf = self.n_fock cops = [] for i in range(self.n): cops.append(np.sqrt(γ_down) * _embed(qt.sigmam(), i, self.n, nf)) cops.append(np.sqrt(γ_up) * _embed(qt.sigmap(), i, self.n, nf)) J_minus = sum(_embed(qt.sigmam(), i, self.n, nf) for i in range(self.n)) cops.append(np.sqrt(self.cfg.gamma_base * 0.5) * J_minus) return cops # ── Measurement operator for homodyne ───────────────────────────────────── def _make_homodyne_operator(self) -> qt.Qobj: law = self.cfg.homodyne_operator if law == "Jx": return sum(0.5 * self._embed(qt.sigmax(), i) for i in range(self.n)) elif law == "Jy": return sum(0.5 * self._embed(qt.sigmay(), i) for i in range(self.n)) elif law == "Jz": return sum(0.5 * self._embed(qt.sigmaz(), i) for i in range(self.n)) else: raise ValueError(f"Unknown homodyne_operator: '{law}'") # ── Evolution ───────────────────────────────────────────────────────────── def evolve(self, reset: bool = True) -> None: """ Run time evolution from t=0 to cfg.t_end. Backend selection (in priority order): 1. Homodyne SME (if homodyne_enabled=True) 2. JAX/Diffrax (if use_jax=True) 3. QuTiP mesolve (default) """ if reset: self.current_state = self.initial_state for lst in self.metrics.values(): lst.clear() self.times.clear() tlist = np.linspace(0, self.cfg.t_end, self.cfg.n_steps) if self.cfg.homodyne_enabled: self._evolve_homodyne(tlist) elif self.cfg.use_jax: self._evolve_jax(tlist) else: self._evolve_qutip(tlist) def _evolve_qutip(self, tlist: np.ndarray) -> None: opts = qt.Options(nsteps=int(1e5), store_states=True) result = qt.mesolve( self.H, self.current_state, tlist, self.c_ops, [], options=opts ) self.times = list(tlist) for rho in result.states: self.current_state = rho self._record_metrics(rho) def _evolve_jax(self, tlist: np.ndarray) -> None: from core.jax_integrator import integrate result = integrate(self.H, self.current_state, tlist, self.c_ops) self.times = list(tlist) for rho in result.states: self.current_state = rho self._record_metrics(rho) def _evolve_homodyne(self, tlist: np.ndarray) -> None: from measurement.HomodyneSME import HomodyneSME from measurement.BayesianFeedback import BayesianFeedback L_meas = self._make_homodyne_operator() sme = HomodyneSME( H = self.H, c_ops = self.c_ops, meas_op = np.sqrt(self.cfg.gamma_base) * L_meas, efficiency = self.cfg.homodyne_efficiency, n_traj = 50, seed = self.cfg.seed or 42, use_jax = self.cfg.use_jax, ) hres = sme.run(self.current_state, tlist) self.times = list(tlist) self.photocurrent = hres.photocurrent for rho in hres.states: self.current_state = rho self._record_metrics(rho) if self.cfg.bayesian_feedback: fb = BayesianFeedback( H_drift = self.H, L_meas = np.sqrt(self.cfg.gamma_base) * L_meas, c_ops = self.c_ops, efficiency = self.cfg.homodyne_efficiency, feedback_law = "wiseman_milburn", n_qubits = self.n, ) mean_pc = hres.photocurrent.mean(axis=0) self.estimated_states, _ = fb.run( self.current_state, tlist, mean_pc ) # ── Metric recording ────────────────────────────────────────────────────── def _record_metrics(self, rho: qt.Qobj) -> None: self.metrics["coherence"].append(self._measure_coherence(rho)) self.metrics["entropy"].append(self._measure_entanglement(rho)) self.metrics["population"].append(float(np.real(qt.expect( self._embed(qt.num(2), 0), rho )))) self.metrics["e_variance"].append(self._energy_variance(rho)) self.metrics["negativity"].append(self._measure_negativity(rho)) self.metrics["qfi"].append(self._quantum_fisher_info_sld(rho)) # ── Observables ─────────────────────────────────────────────────────────── def _measure_coherence(self, rho: qt.Qobj) -> float: dm = rho.full() return float(np.sum(np.abs(dm)) - np.sum(np.abs(np.diag(np.diag(dm))))) def _measure_entanglement(self, rho: qt.Qobj) -> float: rho_q = self._trace_cavity(rho) n_a = self.n // 2 rho_A = rho_q.ptrace(list(range(n_a))) return float(qt.entropy_vn(rho_A, base=2)) def _energy_variance(self, rho: qt.Qobj) -> float: e1 = float(np.real(qt.expect(self.H, rho))) e2 = float(np.real(qt.expect(self.H * self.H, rho))) return e2 - e1 ** 2 def _measure_negativity(self, rho: qt.Qobj) -> float: rho_q = self._trace_cavity(rho) n_a = self.n // 2 d_a = 2 ** n_a d_b = 2 ** (self.n - n_a) dm = rho_q.full().reshape(d_a, d_b, d_a, d_b) pt = dm.transpose(2, 1, 0, 3).reshape(d_a * d_b, d_a * d_b) eigs = np.linalg.eigvalsh(pt) neg = (np.sum(np.abs(eigs)) - 1.0) / 2.0 return float(np.log2(1.0 + 2.0 * max(neg, 0.0))) def _quantum_fisher_info_sld(self, rho: qt.Qobj) -> float: """ Exact SLD Quantum Fisher Information. QFI = 2 Σ_{a,b: λ_a+λ_b>0} |⟨a|G|b⟩|² (λ_a−λ_b)² / (λ_a+λ_b) """ rho_q = self._trace_cavity(rho) Jz = sum(0.5 * self._embed_qubit_only(qt.sigmaz(), i) for i in range(self.n)) vals, vecs = rho_q.eigenstates() qfi = 0.0 for a in range(len(vals)): for b in range(len(vals)): denom = vals[a] + vals[b] if denom < 1e-12: continue mel = (vecs[a].dag() * Jz * vecs[b]).full()[0, 0] qfi += 2.0 * (vals[a] - vals[b]) ** 2 / denom * np.abs(mel) ** 2 return float(np.real(qfi)) def _trace_cavity(self, rho: qt.Qobj) -> qt.Qobj: if self.n_fock == 1: return rho return rho.ptrace(list(range(self.n))) def qubit_populations(self, rho: Optional[qt.Qobj] = None) -> List[float]: if rho is None: rho = self.current_state rho_q = self._trace_cavity(rho) pops = [] for i in range(self.n): rho_i = rho_q.ptrace([i]) pops.append(float(np.real(rho_i[1, 1][0, 0]))) return pops # ── Control ─────────────────────────────────────────────────────────────── def apply_rabi_pulse( self, target_qubit : int = 0, omega_drive : float = 1.0, duration : float = np.pi, ) -> Dict: H_drive = self.H + omega_drive * self._embed(qt.sigmax(), target_qubit) tlist = np.linspace(0, duration, 200) opts = qt.Options(nsteps=int(1e5)) result = qt.mesolve(H_drive, self.current_state, tlist, self.c_ops, [], options=opts) self.current_state = result.states[-1] ideal_U = (-1j * np.pi / 2 * qt.sigmax()).expm() rho_q = self._trace_cavity(self.current_state).ptrace([target_qubit]) rho_ideal= qt.ket2dm(ideal_U * qt.basis(2, 1)) fidelity = float(np.real(qt.fidelity(rho_q, rho_ideal) ** 2)) return { "gate_fidelity" : fidelity, "final_population": float(np.real(rho_q[1, 1][0, 0])), } def optimize_gate( self, target_gate_name : str = "CNOT", T : float = None, n_steps : int = None, ) -> Dict: from core.OptimalControl import GRAPEOptimizer T = T or self.cfg.qoc_T n_steps = n_steps or self.cfg.qoc_n_steps qubits = self.cfg.target_qubits opt = GRAPEOptimizer( H_drift = self.H, H_controls = [self._embed(qt.sigmax(), i) for i in qubits], c_ops = self.c_ops, n_steps = n_steps, T = T, target_gate_name = target_gate_name, target_qubits = qubits, n_total = self.n, n_fock = self.n_fock, method = self.cfg.grape_method, ) return opt.run() # ── Summary ─────────────────────────────────────────────────────────────── def summary(self) -> Dict: if not self.metrics["coherence"]: return {"error": "No evolution data. Call evolve() first."} result = { "n_sites" : self.n, "n_protofilaments" : self.cfg.n_protofilaments, "final_coherence" : self.metrics["coherence"][-1], "final_entropy_bits": self.metrics["entropy"][-1], "final_negativity" : self.metrics["negativity"][-1], "final_qfi_sld" : self.metrics["qfi"][-1], "peak_e_variance" : max(self.metrics["e_variance"]), "qubit_populations" : self.qubit_populations(), "homodyne_enabled" : self.cfg.homodyne_enabled, "jax_backend" : self.cfg.use_jax, } if self.cfg.biological_units: H_exp = float(np.real(qt.expect(self.H, self.current_state))) result["hamiltonian_energy_GHz"] = H_exp * MEV_TO_GHZ result["hamiltonian_energy_wn"] = H_exp * MEV_TO_WAVENUMBER return result

V5 Improvement Summary

1. JAX/Diffrax GPU Integrator (core/jax_integrator.py)

jax_integrator.integrate() implements the full Lindblad master equation as a JAX-jit-compiled Diffrax ODE. The density matrix is vectorised as a flat real array and the RHS is compiled once at runtime. On NVIDIA hardware, this replaces QuTiP's single-threaded CPU mesolve with a fully GPU-accelerated adaptive step-size solver. The interface is identical to QuTiP so QuantumLatticeProcessor.evolve() selects it transparently via cfg.use_jax=True.

2. Full 2D Cylindrical Lattice (lattice/CylindricalLattice.py)

CylindricalLattice(N, n_pf=13) builds the complete N × 13 qubit grid with proper 3D helical positions. It generates all nearest-neighbour bonds (axial + angular), respects periodic_axial and periodic_angular boundary conditions independently, and exposes a cached displacement matrix used by the pair-sum loop in QuantumLatticeProcessor._H_long_range_dipole().

3. Minimum-Image Convention (lattice/MinImageConvention.py, physics_constants.py)

minimum_image_displacement() replaces the V4 naïve r_lattice = (j - i) approximation. The V4 approach was only correct for the single wrap-around bond; all longer-range pair distances were wrong under periodic boundaries. V5 applies the standard minimum-image formula to every pair, with separate box lengths for the axial and angular directions and proper angular minimum-image for the discrete protofilament index.

4. Continuous Homodyne Measurement (measurement/HomodyneSME.py)

HomodyneSME implements the diffusive (homodyne) unravelling of the Lindblad equation via both QuTiP smesolve and a custom JAX Euler-Milstein loop. The ensemble-averaged state and the raw photocurrent record I(t_k) are both returned. The photocurrent is consumed by the Bayesian feedback module.

5. Bayesian Feedback Controller (measurement/BayesianFeedback.py)

BayesianFeedback maintains an online state estimate ρ̂(t) from the homodyne photocurrent and optionally closes the loop with a feedback Hamiltonian. Two built-in laws are provided: Wiseman-Milburn phase-space feedback and linear-quadratic-Gaussian (LQG) control on the Bloch z-component.

6. Literature Validation (analytics/LiteratureValidation.py)

LiteratureValidator compares simulation observables (coherence length, resonance frequencies, decoherence rate) against digitised reference values from Bandyopadhyay (2013) and Craddock/Tuszynski (2017). A normalised chi-squared figure of merit is reported; a simulation passes if χ²/dof < 2.
